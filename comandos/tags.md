## Crear tag
```
git tag tag-name
```

## Borrar tag
```
git tag -d tag-name
```

## Crear tag
```
git tag -a v1.0.0 -m "tag initial"
```

## Ver todos los tag
```
git tag
```

## Crear tag con commit
```
git tag -a v0.1.0 #-commit -m "tag initial x2"
```

## Ver detalle del tag
```
git show v0.1.0
```

## Subir los tags
```
git push --tags
```
